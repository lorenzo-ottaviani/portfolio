My Calculator

Avec le projet My Calculator, on a créé une simple calculette qui s'affiche dans votre terminal. Elle vous permet de réaliser les sept opérations basiques codées par le langage Python : l'addition, la soustraction, la multiplication, les divisions simple et euclidienne (avec reste), le calcul de puissances et le modulo d'un nombre (le reste de la division euclidienne). Un menu, lancé dès l'ouverture du programme, permet de choisir si l'on souhaite effectuer une opération avec deux nouveaux nombres, utiliser le résultat précédent dans notre calcul ou tout simplement afficher l'historique de nos opérations qui a été sauvegardé dans un fichier JSON par notre programme.

Langage utilisé : Python.

Budget Buddy

Budget Buddy est une application de gestion financière conçue avec le module customtkinter de Python reliée à une base de données MySQL. À son lancement, l'application vérifie si la base de données existe déjà sur votre ordinateur, sinon elle va la créer automatiquement de zéro. Pour utiliser notre application, vous devrez créer un compte client. Lors de l'enregistrement de vos données personnelles dans notre base, votre mot de passe et votre email seront cryptés pour plus de sécurité. Une fois connecté, Budget Buddy, vous permettra de gérer facilement vos comptes, relire l'historique de vos transactions et en effectuer de nouvelles.

Langages utilisés : Python & SQL.
